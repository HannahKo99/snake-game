# 霓虹戰術貪吃蛇 (Neon Tactical Snake) - 專案企劃書

## 1. 專案概述 (Project Overview)
本專案旨在打造一款結合現代視覺風格與動作要素的網頁遊戲。區別於傳統貪吃蛇，本作引入了**「子彈時間」**、**「能量系統」**與**「連擊機制」**，解決了傳統玩法後期容錯率低且枯燥的痛點，提供更具深度與爽快感的遊戲體驗。

## 2. 檔案結構與功能詳解 (File Structure & Functions)

本專案採用模組化設計，將不同職責的代碼分離，易於維護與擴充。

### 📄 `index.html` - 遊戲入口與介面架構
*   **作用**：網頁的主骨架，負責載入所有資源。
*   **主要功能**：
    *   **Canvas 容器**：定義遊戲繪圖區域。
    *   **UI 層 (Overlay)**：包含分數板、能量條、連擊提示、開始/結束畫面。
    *   **資源載入**：依序引入 CSS 樣式表與所有 JavaScript 遊戲模組。

### 🎨 `style.css` - 視覺設計與動畫系統
*   **作用**：定義遊戲的「Cyberpunk 霓虹風格」。
*   **主要功能**：
    *   **視覺特效**：使用 CSS `box-shadow` 與 `text-shadow` 創造發光質感。
    *   **動畫效果**：定義 UI 的淡入淡出、按鈕懸停效果及連擊文字的縮放動畫。
    *   **響應式佈局**：確保遊戲介面在不同螢幕尺寸下皆能正確置中顯示。

### ⚙️ `game.js` - 遊戲核心控制器 (Game Engine)
*   **作用**：遊戲的「大腦」，負責協調所有模組運作。
*   **主要功能**：
    *   **遊戲循環 (Game Loop)**：使用 `requestAnimationFrame` 驅動遊戲，並計算 `deltaTime` 以確保動畫流暢。
    *   **狀態管理**：管理遊戲狀態（開始、進行中、結束）、分數計算、最高分存儲。
    *   **時間控制**：核心的「子彈時間」邏輯在此實作，透過調整 `timeScale` 改變遊戲速度。
    *   **UI 更新**：即時將遊戲數據（分數、能量）同步到介面上。

### 🐍 `Snake.js` - 玩家角色邏輯
*   **作用**：定義主角「蛇」的行為與屬性。
*   **主要功能**：
    *   **移動系統**：基於網格 (Grid-based) 的移動邏輯，並處理身體的跟隨。
    *   **成長機制**：吃到食物後身體變長。
    *   **碰撞檢測**：判斷是否撞牆或撞到自己（遊戲結束條件）。
    *   **繪製渲染**：繪製蛇的身體，並為頭部添加特殊的發光效果。

### 🍎 `Food.js` - 食物與道具系統
*   **作用**：管理遊戲中可收集物件的生成與渲染。
*   **主要功能**：
    *   **隨機生成**：在網格上隨機尋找空位生成食物，避免生成在蛇身上。
    *   **多樣化類型**：
        *   **普通食物 (Cyan)**：增加長度與分數。
        *   **能量球 (Magenta)**：大量恢復能量，用於發動技能。
        *   **加分道具 (Yellow)**：提供高額分數獎勵。
    *   **動態效果**：實作食物的呼吸燈閃爍動畫。

### ✨ `Particle.js` - 粒子特效系統
*   **作用**：負責遊戲中的視覺回饋 (Juice)，提升打擊感。
*   **主要功能**：
    *   **粒子生成**：在特定事件（如吃食物、撞擊）發生位置生成大量微小粒子。
    *   **物理模擬**：每個粒子擁有獨立的速度、壽命與衰減邏輯。
    *   **渲染優化**：自動清理生命週期結束的粒子，維持效能。

### 🎮 `Input.js` - 輸入控制模組
*   **作用**：監聽玩家的操作指令。
*   **主要功能**：
    *   **方向控制**：支援方向鍵與 WASD，並設有輸入隊列 (Input Queue) 防止快速按鍵導致的邏輯衝突。
    *   **技能觸發**：監聽空白鍵 (Space) 的按下與放開，控制「子彈時間」的開關。

## 3. 核心特色與競爭優勢 (Key Features)

1.  **子彈時間 (Bullet Time)**
    *   **痛點解決**：傳統貪吃蛇速度變快後極易失誤。
    *   **解決方案**：玩家可消耗能量讓時間變慢，獲得思考與反應的空間，大幅提升容錯率與操作上限。

2.  **能量資源管理 (Resource Management)**
    *   **深度玩法**：不僅僅是吃食物變長，玩家還需要尋找「能量球」來維持技能的使用，增加了策略維度。

3.  **連擊系統 (Combo System)**
    *   **爽快感**：鼓勵玩家快速連續進食，獲得倍率分數，創造高風險高回報的挑戰體驗。

4.  **極致的視聽體驗**
    *   **風格**：採用高對比度的霓虹配色，配合粒子爆炸特效，帶來強烈的視覺衝擊。
